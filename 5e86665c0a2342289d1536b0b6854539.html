Introduction
This is the second blog post in the series of blog posts on kernels. In the first part I introduced the functional analysis background for kernel theory. I highly recommend you read it before continuing. I will frequently refer to it and use the same notation. In this blog post I aim to introduce the fundamental theorems like Mercer's theorem and Representer theorem.
Characterization of Reproducing Kernels
We already defined the notion of reproducing kernels for an RKHS (Definition 20). We now turn our attention to obtaining necessary and sufficient conditions for a function K ⁣:X×X→CK \colon \mathcal{X} \times \mathcal{X} \to \mathbb{C}K:X×X→C﻿ to be the reproducing kernel for some RKHS. But first we must define kernel functions.
Definition 21: Let A=(ai,j)A = (a_{i,j})A=(ai,j​)﻿ be an n×nn \times nn×n﻿ complex matrix. Then AAA﻿ is called positive if for every α1,…,αn∈C\alpha_1, \ldots, \alpha_n \in \mathbb{C}α1​,…,αn​∈C﻿ we have
∑i,j=1nαi‾αjai,j≥0.\sum_{i,j = 1}^{n} \overline{\alpha_i}\alpha_j a_{i,j} \geq 0.i,j=1∑n​αi​​αj​ai,j​≥0.
We denote this by A≥0.A \geq 0.A≥0.﻿
Note that if we define a vector x∈Cnx \in \mathbb{C}^nx∈Cn﻿ to be such that its ithi^{\text{th}}ith﻿ component is αi\alpha_iαi​﻿, then the condition above can rewritten as
⟨Ax,x⟩≥0.\left\langle Ax,x \right\rangle \ge 0.⟨Ax,x⟩≥0.
Also note that if A≥0A \geq 0A≥0﻿ then A=A∗A = A^*A=A∗﻿, where A∗A^*A∗﻿ denotes the Hermitian matrix (also known as the self-adjoint matrix), A⊤‾.\overline{A^\top}.A⊤.﻿ Therefore, positivity gives self-adjoint property for free if we are dealing with complex matrices. Things aren't so elegant for real matrices. For the real case we need to explicitly state that the matrix AAA﻿ is also symmetric apart from what's stated in the definition above. Therefore, we often use the following as the definition of positive matrices:
Definition 21': An n×nn \times nn×n﻿ matrix AAA﻿ is positive, in symbols A≥0A \geq 0A≥0﻿, if it is self-adjoint and if ⟨Ax,x⟩≥0\left\langle Ax,x \right\rangle \ge 0⟨Ax,x⟩≥0﻿ for all x∈Cn.x \in \mathbb{C}^n.x∈Cn.﻿
Positive matrices are also alternatively called positive semidefinite or nonnegative matrices.
The following lemma connects the concept of positive matrices to its eigenvalues.
Lemma 2: A matrix A≥0A \geq 0A≥0﻿ if and only if A=A∗A = A^*A=A∗﻿ and every eigenvalue of AAA﻿ is nonnegative.
Proof:
Let us first suppose A≥0A \geq 0A≥0﻿, then A=A∗A = A^*A=A∗﻿ by definition. Now if λ\lambdaλ﻿ is an eigenvalue of AAA﻿ and vvv﻿ is an eigenvector of AAA﻿ corresponding to the eigenvalue λ\lambdaλ﻿, we have
0≤⟨Av,v⟩=⟨λv,v⟩=λ⟨v,v⟩.0 \leq \left\langle Av,v \right\rangle = \left\langle \lambda v,v \right\rangle = \lambda \left\langle v,v \right\rangle.0≤⟨Av,v⟩=⟨λv,v⟩=λ⟨v,v⟩.
Since ⟨v,v⟩≥0\left\langle v,v \right\rangle \ge 0⟨v,v⟩≥0﻿, λ\lambdaλ﻿ is a nonnegative number.
For the other side, find an orthonormal basis v1,…,vnv_1, \ldots, v_nv1​,…,vn​﻿ consisting of eigenvectors of AAA﻿ (it exists by the spectral theorem). Let viv_ivi​﻿ be the eigenvector corresponding to the eigenvalue λi.\lambda_i.λi​.﻿ Then for any x=∑i=1nαivix = \sum_{i=1}^n \alpha_i v_ix=∑i=1n​αi​vi​﻿ we have ⟨Ax,x⟩=∑i=1nλi∣αi∣2.\left\langle Ax,x \right\rangle = \sum_{i=1}^n \lambda_i \lvert \alpha_i\rvert^2.⟨Ax,x⟩=∑i=1n​λi​∣αi​∣2.﻿ Since λi≥0\lambda_i \geq 0λi​≥0﻿, ⟨Ax,x⟩≥0\left\langle Ax,x \right\rangle \geq 0⟨Ax,x⟩≥0﻿ and AAA﻿ must be positive. □\square□﻿
We are now ready to define a kernel function.
Definition 22: Let X\mathcal{X}X﻿ be a set, then K:X×X→CK : \mathcal{X} \times \mathcal{X} \to \mathbb{C}K:X×X→C﻿ is called a kernel function if for every n∈Nn \in \Nn∈N﻿ and for every choice of {x1,…,xn}⊆X\{x_1, \ldots, x_n\} \subseteq \mathcal{X}{x1​,…,xn​}⊆X﻿, the matrix (K(xi,xj))≥0.(K(x_i, x_j)) \geq 0.(K(xi​,xj​))≥0.﻿ We will use the notation K≥0K \geq 0K≥0﻿ to denote that the function KKK﻿ is a kernel function.
Kernel functions are alternatively called positive definite functions or positive semidefinite functions.
Definition 23: Given a kernel function K:X×X→CK : \mathcal{X} \times \mathcal{X} \to \mathbb CK:X×X→C﻿ and points x1,…,xn∈Xx_1, \ldots, x_n \in \mathcal{X}x1​,…,xn​∈X﻿, the n×nn \times nn×n﻿ matrix (K(xi,xj))(K(x_i, x_j))(K(xi​,xj​))﻿, denoted K\mathbf{K}K﻿, is called the Gram matrix of KKK﻿ with respect to x1,…,xn.x_1, \ldots, x_n.x1​,…,xn​.﻿
Some examples follow:
Examples
Linear kernels: When X=Rd\mathcal{X} = \R^dX=Rd﻿, we can define the linear kernel function as
K(x,y)=⟨x,y⟩.K(x,y) = \left\langle x,y\right\rangle.K(x,y)=⟨x,y⟩.
It is clearly a symmetric function of its arguments, and hence self-adjoint. To prove positivity, let x1,…,xn∈Rdx_1, \ldots, x_n \in \R^dx1​,…,xn​∈Rd﻿ be an arbitrary collection of points, and consider its gram matrix K\mathbf{K}K﻿, i.e., Ki,j=K(xi,xj)=⟨xi,xj⟩.\mathbf{K}_{i,j} = K(x_i, x_j) = \left\langle x_i,x_j\right\rangle.Ki,j​=K(xi​,xj​)=⟨xi​,xj​⟩.﻿ Then for any α∈Rn\alpha \in \R^nα∈Rn﻿, we have
⟨Kα,α⟩=α⊤K⊤α=α⊤Kα=∑i,j=1nαiαj⟨xi,xj⟩=∥∑i=1nαixi∥2≥0.\left\langle \mathbf{K} \alpha, \alpha \right\rangle = \alpha^\top \mathbf{K}^\top \alpha = \alpha^\top \mathbf{K} \alpha = \sum_{i,j = 1}^n \alpha_i \alpha_j \left\langle x_i, x_j \right\rangle = \left\lVert \sum_{i=1}^n \alpha_i x_i \right\rVert^2 \geq 0.⟨Kα,α⟩=α⊤K⊤α=α⊤Kα=i,j=1∑n​αi​αj​⟨xi​,xj​⟩=∥∥​i=1∑n​αi​xi​∥∥​2≥0.
Polynomial kernels: A natural generalization of the linear kernel on Rd\R^dRd﻿ is the homogeneous polynomial kernel
K(x,y)=(⟨x,y⟩)mK(x, y) = (\left\langle x,y\right\rangle)^mK(x,y)=(⟨x,y⟩)m

K(x,y)=(∑i=1dxiyi)m.K(x,y) = \left( \sum_{i=1}^d x_i y_i \right)^m.K(x,y)=(i=1∑d​xi​yi​)m.
This will have D=(m+d−1m)D = \binom{m+d-1}{m}D=(mm+d−1​)﻿ monomials, so to simplify the analysis let's take m=2.m=2.m=2.﻿ Then
K(x,y)=∑i=1dxi2yi2+2∑i<jxixjyiyj.K(x,y) = \sum_{i=1}^d x_i^2 y_i^2 + 2 \sum_{i < j} x_i x_j y_i y_j.K(x,y)=i=1∑d​xi2​yi2​+2i<j∑​xi​xj​yi​yj​.
In this case D=(d+1d)=d+(d2).D = \binom{d+1}{d} = d + \binom{d}{2}.D=(dd+1​)=d+(2d​).﻿ Define a mapping Φ ⁣:Rd→RD\Phi\colon \R^d \to \R^DΦ:Rd→RD﻿ such that
Φ(x)=[x12,…,xd2,2x1x2,…,2xd−1xd]⊤.\Phi(x) = [x_1^2, \ldots, x_d^2, \sqrt{2}x_1 x_2, \ldots, \sqrt{2} x_{d-1} x_d ]^\top.Φ(x)=[x12​,…,xd2​,2​x1​x2​,…,2​xd−1​xd​]⊤.

K(x,y)=⟨Φ(x),Φ(y)⟩.K(x,y) = \left\langle \Phi(x),\Phi(y)\right\rangle.K(x,y)=⟨Φ(x),Φ(y)⟩.
Following the same argument as the first example, we can verify that the gram matrix thus formed is indeed positive.
The mapping x↦Φ(x)x \mapsto \Phi(x)x↦Φ(x)﻿ is often referred to as a feature map. We see that dealing with elements in the feature space, i.e., the codomain of Φ\PhiΦ﻿, is computationally expensive. The relation K(x,y)=⟨Φ(x),Φ(y)⟩K(x,y) = \left\langle \Phi(x),\Phi(y)\right\rangleK(x,y)=⟨Φ(x),Φ(y)⟩﻿ allows us compute the inner products using the kernel function instead of actually taking the inner product in a very high dimensional space. We will see that this "kernel trick" holds for many kernel functions when we discuss Mercer's theorem.

Equivalence between kernel function and reproducing kernel
Let us return to the characterization of reproducing kernels. We will now prove that a function is a kernel function if and only if there is an RKHS for which it is the reproducing kernel. At this point recall Theorem-3 which states that an RKHS admits a unique reproducing kernel.
Theorem 4: Let X\mathcal{X}X﻿ be a set and let H\mathcal{H}H﻿ be an RKHS on X\mathcal{X}X﻿ with reproducing kernel K.K.K.﻿ Then KKK﻿ is a kernel function.
Proof:
For some arbitrary n∈Nn \in \Nn∈N﻿ fix some arbitrary collection x1,…,xn∈Xx_1, \ldots, x_n \in \mathcal{X}x1​,…,xn​∈X﻿ and α∈Cn.\alpha \in \mathbb{C}^n.α∈Cn.﻿ Then if we denote by K\mathbf{K}K﻿ the gram matrix of KKK﻿ with respect to x1,…,xnx_1, \ldots, x_nx1​,…,xn​﻿, we have
⟨Kα,α⟩=∑i,j=1nαi‾αjK(xi,xj)=∑i,j=1nαi‾αj⟨kxj,kxi⟩=∥∑i=1nαikxi∥2≥0.\left\langle \mathbf{K} \alpha,\alpha \right\rangle = \sum_{i,j = 1}^n \overline{\alpha_i}\alpha_j K(x_i, x_j) = \sum_{i,j = 1}^n \overline{\alpha_i}\alpha_j \left\langle k_{x_j},k_{x_i} \right\rangle = \left\lVert \sum_{i=1}^n \alpha_i k_{x_i} \right\rVert^2 \geq 0. ⟨Kα,α⟩=i,j=1∑n​αi​​αj​K(xi​,xj​)=i,j=1∑n​αi​​αj​⟨kxj​​,kxi​​⟩=∥∥​i=1∑n​αi​kxi​​∥∥​2≥0.
And thus KKK﻿ is a kernel function. □\square□﻿
What does it mean in the above proof if we have an equality? That is, if ⟨Kα,α⟩=0\left\langle \mathbf{K} \alpha,\alpha \right\rangle = 0⟨Kα,α⟩=0﻿? This happens if and only if ∥∑i=1nαikxi∥=0.\left\lVert \sum_{i=1}^n \alpha_i k_{x_i} \right\rVert = 0.∥∑i=1n​αi​kxi​​∥=0.﻿ But this means that for every f∈Hf \in \mathcal{H}f∈H﻿ we have ∑i=1nαi‾f(xi)=⟨f,∑iαikxi⟩=0.\sum_{i=1}^n \overline{\alpha_i} f(x_i) = \left\langle f,\sum_i \alpha_i k_{x_i} \right\rangle = 0.∑i=1n​αi​​f(xi​)=⟨f,∑i​αi​kxi​​⟩=0.﻿ Thus, in this case there is an equation of linear dependence between the values of every function in H\mathcal{H}H﻿ at this finite set of points.
Now let us state the converse of Theorem-4. It is a deep result in RKHS theory known as the Moore–Aronszajn theorem.
Theorem 5 [Moore–Aronszajn theorem]: Let X\mathcal{X}X﻿ be a set and let K:X×X→CK : \mathcal{X} \times \mathcal{X} \to \mathbb{C}K:X×X→C﻿ be a kernel function, then there exists a reproducing kernel Hilbert space H\mathcal{H}H﻿ of functions on X\mathcal{X}X﻿ such that KKK﻿ is the reproducing kernel of H.\mathcal{H}.H.﻿
For a proof see here on Wikipedia.
In light of these two theorems we have the following notation.
Definition 24: Given a kernel function K:X×X→CK : \mathcal{X} \times \mathcal{X} \to \mathbb{C}K:X×X→C﻿, we let HK\mathcal{H}_KHK​﻿ denote the unique RKHS with the reproducing kernel K.K.K.﻿
It is not an easy problem to start with a kernel function KKK﻿ on some set X\mathcal{X}X﻿ and give a concrete description of HK.\mathcal{H}_K.HK​.﻿ I will not be discussing this here, but there are plenty of resources available where you can see this problem getting discussed. I recommend Paulsen and Raghupathi's An Introduction to the Theory of Reproducing Kernel Hilbert Spaces.
Mercer's theorem
Recall the spectral theorem for finite-dimensional vector spaces: A linear operator T ⁣:V→VT\colon V \to VT:V→V﻿ for some finite dimensional vector space VVV﻿ on C\mathbb{C}C﻿ is normal, i.e., TT∗=T∗TT T^* = T^* TTT∗=T∗T﻿, if and only if VVV﻿ has an orthonormal basis consisting of eigenvectors of T.T.T.﻿ This implies that if U=[v1,…,vn]\mathbf{U} = [v_1, \ldots, v_n]U=[v1​,…,vn​]﻿ is a unitary matrix containing the ithi^{\text{th}}ith﻿ eigenvector in its ithi^{\text{th}}ith﻿ column and Λ=diag(λ1,…,λn)\mathbf{\Lambda} = \text{diag}(\lambda_1, \ldots, \lambda_n)Λ=diag(λ1​,…,λn​)﻿ is a diagonal matrix containing the corresponding eigenvalues, then if K\mathbf{K}K﻿ is a normal matrix then it can written as
K=UΛU⊤=∑i=1nλivivi⊤.\mathbf{K} = \mathbf{U} \mathbf{\Lambda} \mathbf{U}^\top = \sum_{i=1}^n \lambda_i v_i v_i^\top.K=UΛU⊤=i=1∑n​λi​vi​vi⊤​.
